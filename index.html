Basic HTML Tags
HTML Tags Notes Output/Examples 
<DOCTYPE! html>
 This doc type tag is important as it will always default to the most current HTML standard. No output to display but it will always be the first code in an HTML file. 
<html> </html>
 This tag is the root element of an HTML page because it adds the HTML structure to your webpage. No output to display but it will always be the second code in an HTML file. 
<head> </head>
 This tag contains meta-information about the HTML document such as the title of the page, links to CSS, and more. No output to display but it will always be the third code in an HTML file. 
<body> </body>
 This tag contains the content of the HTML document such as text, images, tables, and more. No output to display but it will always be the fourth code in an HTML file. 
<a href="URL or ./relative path" 
 target="_blank"/>Insert text to display.</a>
 Links are unique in that they have a self-closing tag. In this coding, I used an anchor tag that does require a closing tag, but I did this to show you how to utilize this coding as you are likely to use it. They require URLs and you can include target attributes to tell the browser how to open the link. Link to Vecteezy who provided the free HTML video you can see below.

<p> <p>
 Paragraph tags are used to create blocks of text. Paragraphs are great for chunks of text. Our next code will be better for smaller pieces of text. 
<span> </span>
 Span tags are great for smaller pieces of text. Say one sentence that you may want to call attention to out of a paragraph. Out of a paragraph, you might want to call attention to something, such as pricing.Aside tags may also be an option you want to consider. 
<ul> </ul>
 Unordered list tags are necessary if you want to display a bulleted list of items. In order to add your bulleted list, you must include your list items with the <li> tags.

<ol> </ol>
 Ordered list tags are necessary if you want to display a numbered list of items. In order to add your numbered list, you must include your list items with the <li> tags.

<img src="URL or ./relative path" alt="insert
description" width="300" height="240" />
 Image tags are necessary if you want to display an image on your webpage. You must include the source of the image and an alt description for accessibility purposes.
